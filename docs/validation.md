# Validation Usage

## Validate Myanmar Phone Number

```php
$request->validate([
    'phone' => ['required', 'myanmarPhoneNumber'],
]);
```

## Validate MPT

```php
$request->validate([
    'phone' => ['required', 'mpt'],
]);
```

## Validate Ooredoo

```php
$request->validate([
    'phone' => ['required', 'ooredoo'],
]);
```
